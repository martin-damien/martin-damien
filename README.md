### About writing code as a hobby

After spending so many years building web applications professionally, nowadays I prefer learning older technologies.
I find the DOS/Windows 9x era very interesting : having to work with limited hardware resources, old-school programming languages and of course nostalgia.

I started using PCs with Windows 3.11 that didn't have enough RAM to run Duke Nukem 3D, and I learned programming with Visual Basic 6.

That's why most of the code you'll find here targets DOS... (C or Pascal) or Delphi/VB6-style applications built with Lazarus.

I'm currently interested in:

- C / Free Pascal / Lazarus
- DOS programming
- Building small native desktop applications
- Learning how software was written before modern frameworks
